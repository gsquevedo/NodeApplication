## Objetivo
- Aprimorar conhecimentos em Nodejs

## Funcionalidades
- Criação de users
- Criação de tags
- Criação de elogios
- Autenticação de users
- Listagem de users, tags e elogios

## Tecnologias
- Typescript
- Nodejs
- Express
- JSONWebToken
- Sqlite

## Como compilar
- Clone o repositório
- Execute ``yarn`` 
- Execute ``yarn typeorm migration:run``
- Execute ``yarn dev``
- A aplicação está disponível em ``http://localhost:3000``
